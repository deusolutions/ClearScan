# Интеграционные тесты для ClearScan

- test_integration.py: Проверяет полный сценарий работы — сканирование, сравнение, запись изменений, уведомление через Telegram-бота.
- Используется временная база данных для изоляции тестов.
- Отправка уведомлений мокируется для проверки вызова.

## Как запустить

```bash
pytest tests/integration/
```

## Что проверяется
- Корректность взаимодействия scanner.py, comparator.py, telegram_bot.py
- Корректная запись изменений в scan_history
- Корректная генерация уведомлений о новых/закрытых портах
