# Changelog

## [1.1.0] - 2025-05-20
### Added
- Полностью автоматизированная установка через install.sh (создание пользователя, копирование файлов, настройка прав, systemd, запуск сервисов).
- Разделение systemd-сервисов: dashboard и Telegram-бот теперь отдельные юниты.
- install.sh теперь копирует проект в /opt/clearscan и всё настраивает автоматически.
- README.md обновлён: установка — одной командой, только для Linux.
- Пример конфигурации создаётся корректно, читается в nano.

### Changed
- Улучшена безопасность: сервисы запускаются только от отдельного пользователя clearscan.
- Инструкции по ручной установке и Windows удалены из документации.

### Fixed
- Исправлены права и автоматизация для production-ready эксплуатации.

## [1.0.0] - 2025-05-20
### Added
- Первая стабильная версия ClearScan: автоматизированный сканер подсетей с nmap, хранением результатов в SQLite, уведомлениями через Telegram-бота, веб-дашбордом на Flask, cron/systemd-автоматизацией, логированием, безопасностью, CI/CD, документацией и интеграционными тестами.
